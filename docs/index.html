XAF Converter

Convert Dutch XAF (XML Auditfile Financieel) files to CSV, XLSX, JSON, and Parquet. Self-hosted, private, and runs entirely in Docker.

View on GitHub

Multiple Formats

Export to CSV, XLSX (styled), JSON, or Parquet. One upload, multiple exports.

XAF 3.1 / 3.2 / 4.0

Supports all major XAF versions from any Dutch accounting system.

Self-Hosted & Private

Your financial data never leaves your server. No external calls, no cloud dependencies.

Data Validation

Automatic integrity checks against XAF control totals for debit/credit sums and line counts.

Preview & Search

Browse and search all records before exporting. Select only the data types you need.

Easy Setup

One Docker command to build, one to run. No runtime dependencies beyond Docker.
